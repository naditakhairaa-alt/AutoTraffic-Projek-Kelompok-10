oleh Kelompok 10

Struktur Data yang Diimplementasikan:
1. OOP              - Kelas Kendaraan, Lane, LampuLalin
2. Queue (Prioritas)- Antrian darurat: ambulans/bus didahulukan
3. Stack            - Emergency Undo data kendaraan
4. Single LL        - Log kronologis kendaraan yang sudah lewat
5. Double LL        - Cek jarak antar kendaraan dalam satu jalur
6. Circular LL      - Rotasi lampu hijau (U→T→S→B→U)
7. Hash Table       - Index kendaraan berdasarkan ID
8. BST              - Sorting antrian berdasarkan waktu tunggu
9. Tree (N-ary)     - Klasifikasi kendaraan berdasarkan jenis & status
10. Graph           - Peta rute antar persimpangan
11. Rekursif        - DFS, BST traverse inorder, merge sort, hitung daun tree
12. Sorting         - Merge sort kendaraan berdasarkan waktu tunggu
13. Searching       - Linear + binary search kendaraan
14. File I/O        - Simpan/muat data JSON & laporan TXT
